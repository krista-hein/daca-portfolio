# Nädal 3: SQL JOIN analüüs - Urbanstyle.ltd äriandmete ühendamine

## Ülevaade
Selle nädala väljakutseks oli UrbanStyle killustatud andmete ühendamine. Kuna ärianded asusid erinevates tabelistes (sales, customers, products, inventory), kasutasime SQL JOIN-lauseid, et luua tervikpilt ettevõtte müügist, klientidest ja laoseisust.

## Kausta struktuur
1. **individual** - Minu isiklik panus (Roll A: Müügi ja klientide ühendamine): SQL fail ning tulemuste kuvatõmmised.
2. **team** - Meeskondlik koondanalüüs: ülevaade kadunud klientidest, müügikanalite efektiivsusestja inventuuri kriitilistest vigadest ning strateegilised soovitused UrbanStyle juhtidele.

## Kasutatud tehnoloogiad
* **SQL**: INNER JOIN, LEFT JOIN, LEFT JOIN + IS NULL, Subqueries.
* **Tööriistad**: Supabase, GitHub, Google Slides.
