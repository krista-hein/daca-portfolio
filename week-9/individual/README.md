# Nädal 9 - Roll C - Personalijuhi vaade (People Manager)

## ÜLESANNE:
Minu ülesandeks oli vaadata värbamisprotsessi personalijuhi pilguga ning hinnata, kas kandidaatide tehniline võimekus on tasakaalus pehmete oskuste ja kultuurilise sobivusega.                        

## SISEND: 
Meeskonnakaaslase GitHub portfoolio (README-d, projektikirjeldused, koostöö jäljed) + minu 8-nädalane kogemus koos selle inimesega UrbanStyle meeskonnas.                      

## OODATAV VÄLJUND:
Kirjalik peer review (1 lehekülg):
* Kommunikatsioonioskus: kuidas kirjeldab projekte?
* Probleemilahendus: kas portfooliost paistab loogiline mõtlemine?
* Meeskonnatöö: kas on märke koostööst?
* Kultuuriline sobivus: kas sobib UrbanStyle'i väärtustega?
* 3 tugevust + 2 parandusettepanekut
* Värbamissoovitus: Jah / Lisavestlus / Ei (koos põhjendusega)

---
## TEHTUD TÖÖ:
Selles simulatsioonis vastutasin ma kandidaatide portfoolio ja meeskonnatöö kogemuse hindamise eest pehmete oskuste vaatepunktist.  
**Analüüsisin**:
* **Kommunikatsioonivõimet**: Kuidas tehnilised leiud on tõlgitud ärikeelde.
* **Probleemilahendust**: Iseseisvust ja loogilist mõtlemist takistuste korral.
* **Kultuurilist sobivust**: Vastavust UrbanStyle'i väärtustele (läbipaistvus, andmepõhisus).
