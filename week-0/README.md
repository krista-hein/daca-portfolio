# Nädal 0 kirjeldus 

• Tutvusime omavahel ja saime ülevaate programmist ning 11-nädalasest teekonnast

• Tutvusime lühidalt UrbanStyle.ltd-ga — ettevõte, mille andmeid kogu programmi jooksul analüüsime

• Õppisime AI baasteadmisi — kuidas keelemudelid töötavad ja kuidas neid andmeanalüüsis kasutada

• Tutvusime NotebookLM-iga — meie AI-õppekaaslane, mis genereerib audio ülevaateid materjalidest

• Mõistsime, et AI kasutamine on lahutamatu osa andmeanalüütiku igapäevatööst

• Seadistasime isikliku ja grupitöö jaoks GitHub, Supabase, NotebookLM keskkonnad

• Kohtusime esimest korda oma grupiga ning tegime esimese demo-esitluse, kus näitasime, et GitHub kontod töötavad, Supabase projektid on püsti, NotebookLM genereerib audio ülevaateid ja meeskonnad on moodustatud.
